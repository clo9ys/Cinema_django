# 🎬 Cinema Backend Project — Спринт 1

**Бэкенд-система для онлайн-кинотеатра на базе Django и Django Rest Framework. Проект разработан в рамках учебного курса по веб-разработке.**

##  Технологический стек
- **Фреймворк:** Django 5 + DRF
- **Виртуальное окружение:** uv 
- **База данных:** SQLite   

##  Архитектура проекта
Проект разделен на два основных приложения:
- `movies/` — Управление контентом (Фильмы, Жанры).
- `users/` — Управление профилями, подписками и списками избранного (`Watchlist`).

### Реализованные User Stories
Admin Panel: Кастомизированный интерфейс с фильтрами по жанрам/возрасту и поиском по фильмам.

Database Schema: Реализованы связи Many-to-Many (фильмы-жанры) и One-to-One (профили пользователей).

Data Consistency: Запрет дубликатов в списках избранного (unique_together).

##  Быстрый старт (для разработчиков)

### 1. Подготовка окружения
Убедитесь, что у вас установлен `uv`. Клонируйте репозиторий и установите зависимости:
```bash
uv sync
```
### 2. Настройка переменных окружения
Скопируйте пример настроек и создайте свой локальный файл .env:
```bash
cp .env.example .env
```
Сгенерируйте свой SECRET_KEY для работы с админкой и вставьте в .env
### 3. Миграция базы данных
```bash
make migrate
```
Загрузите предустановленные жанры из фикстуры
```bash
make load_genres
```
### 4. Создание администратора
```bash
make superuser
```
### 5. Запуск сервера
```bash
make run
```
